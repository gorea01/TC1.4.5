HOW TO INSTALL

Per installarla Ã¨ molto facile e funziona sia per SP che per Premium:

SP

  1. Scarica questo zip https://github.com/pupaxxo/TC1.4.5/archive/master.zip
  2. Apri il tuo minecraft.jar che si trova dentro %appdata%\.minecraft\bin
  3. Scompatta tutti gli zip contenuti nel primo nella cartella instMods(quella dentro il primo zip) nel minecraft.jar e ricordati di rimuovere la cartella META-INF (leggi ordine mods sotto)
  4. Copia le cartella config, coremods , ... (quelle contenute dentro lo zip nella cartella minecraft) dentro il tuo .minecraft ( %appdata%\.minecraft\)
Ora dovrebbe andare tutto per qualunque problema esistono le issues (sopra)

Premium

  1. Scarica questo zip https://github.com/pupaxxo/TC1.4.5/archive/master.zip
  2. Crea una nuova instaza in MultiMC
  3. Falla partire e aspetta che scaricare il minecraft.jar
  4. Copia le cartella config, coremods , ... (quelle contenute dentro lo zip nella cartella minecraft) dentro la cartella minecraft ( click su view folder su multimc e dopo click su minecraft)
  5. Copia la cartella instMods dallo zip alla cartella dell'instaza
  6. Con multiMC riordina le mod nell'ordine scritto sotto
Ora dovrebbe andare tutto per qualunque problema esistono le issues (sopra)
